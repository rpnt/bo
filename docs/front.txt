Фротальная часть представляет из себя PWA приложение, которое работает с рест апи ядра.

Фронт включает в себя функции которые реализуют следующие возможности:
1) Ввод данных по заказам из csv файлов
2) Ввод дополнительных ограничений (количества кластеров и их максимальная длина)
3) Запуск подсчета матрицы
4) Вывод прогресса выполнения расчета матрицы
5) Запуск алгоритма кластеризации
6) Вывод прогресса выполнения расчета одномерных кластеров
7) Вывод одномерных кластеров в csv отчет

Желательно проектировать фронт исходя из PWA технологии (т.е. ориентировать интерфейс
 на мобильные устройства и использовать workers)
